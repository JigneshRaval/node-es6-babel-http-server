Example to use ES6 ( using Babel ) in Node.js

Add following to package.json file

"scripts": {
	"build": "babel src -d build",
	"serve": "node build/main"
},

First Run npm run build command in command promt.

Above command will compile/transform ES6 files from "src" folder to ES5 files and will copy it in to build folder

Then run npm serve It will run compiled ES6 file named main.js from build folder

/build/main.js file has http server code and it will start simple node HTTP server

now open the browser and open http://localhost:3000/
